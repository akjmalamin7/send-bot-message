Bot Integration Guide

Slack এবং Telegram বট দিয়ে মেসেজ পাঠানোর নিয়মাবলী

১. Slack Bot Integration

Slack-এ বট তৈরি করে মেসেজ পাঠাতে নিচের ধাপগুলো অনুসরণ করুন:

ধাপ ১: Workspace তৈরি

আপনার একটি একটি অ্যাক্টিভ Slack Workspace থাকতে হবে। না থাকলে slack.com থেকে তৈরি করে নিন।

ধাপ ২: App তৈরি করা

Slack API Dashboard-এ যান এবং "Create New App" বাটনে ক্লিক করে "From scratch" সিলেক্ট করুন। অ্যাপের নাম দিন এবং আপনার Workspace সিলেক্ট করুন।

ধাপ ৩: Scope সেট করা

বাম পাশের মেনু থেকে "OAuth & Permissions"-এ যান। নিচে "Scopes" সেকশনে গিয়ে "Bot Token Scopes"-এ নিচের পারমিশনগুলো যোগ করুন:

  • chat:write (মেসেজ পাঠানোর জন্য)
  • channels:read (চ্যানেল লিস্ট দেখার জন্য)

ধাপ ৪: App ইন্সটল ও টোকেন সংগ্রহ

একই পেজের উপরের দিকে "Install to Workspace" বাটনে ক্লিক করুন। ইন্সটল হয়ে গেলে আপনি একটি "Bot User OAuth Token" পাবেন (এটি xoxb- দিয়ে শুরু হয়)। এই টোকেনটি কপি করে রাখুন।

ধাপ ৫: Postman-এ রিকোয়েস্ট পাঠানো

Slack-এ মেসেজ পাঠানোর জন্য নিচের তথ্যগুলো ব্যবহার করুন:

POST: https://slack.com/api/chat.postMessage

Headers:

Authorization: Bearer YOUR_SLACK_BOT_TOKEN
Content-Type: application/json

Body (JSON):

{
  "channel": "C06JA4DGFssfsNF", 
  "text": "Hello from Slack Bot API!"
}

২. Telegram Bot Integration

Telegram-এ বট তৈরি করে মেসেজ পাঠাতে নিচের ধাপগুলো অনুসরণ করুন:

ধাপ ১: BotFather এর মাধ্যমে বট তৈরি

  • Telegram-এ গিয়ে @BotFather সার্চ করুন।
  • /start বাটনে ক্লিক করুন।
  • /newbot লিখে পাঠান।
  • বটের একটি নাম (Name) দিন।
  • বটের একটি ইউজার নেম (Username) দিন (যা অবশ্যই bot শব্দ দিয়ে শেষ হতে হবে, যেমন: my_test_bot)।

সফলভাবে তৈরি হলে BotFather আপনাকে একটি API Token এবং বটের URL দেবে।

ধাপ ২: Chat ID সংগ্রহ করা

আপনার বটের লিঙ্কে ঢুকে একটি মেসেজ দিন। তারপর নিচের URL-টি ব্রাউজারে ওপেন করে আপনার chat_id খুঁজে বের করুন:

https://api.telegram.org/bot<YOUR_TOKEN>/getUpdates

ধাপ ৩: Postman-এ রিকোয়েস্ট পাঠানো

Telegram-এ মেসেজ পাঠানোর জন্য নিচের নিয়মটি অনুসরণ করুন:

POST: https://api.telegram.org/bot<YOUR_TOKEN>/sendMessage

Headers:

Content-Type: application/json

Body (JSON):

{
    "chat_id": "5204155030",
    "text": "Hello Akjm! I'm bot"
}
নোট: Telegram-এর ক্ষেত্রে টোকেনটি সরাসরি URL-এর মধ্যেই থাকে (/bot<token>/ ফরম্যাটে)। আলাদা করে হেডার না দিলেও চলে, তবে JSON বডি পাঠানোর জন্য Content-Type: application/json হেডার ব্যবহার করা জরুরি।
