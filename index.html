Slack Messenger

Slack Bot সেটআপ নিয়ম

১. Workspace তৈরি: আপনার একটি Slack Workspace আগে থেকে ক্রিয়েট করা থাকতে হবে।
২. App তৈরি করা: api.slack.com/apps এ গিয়ে আপনার Workspace এর জন্য একটি নতুন App ক্রিয়েট করতে হবে।
৩. Scope সেট করা: অ্যাপ ড্যাশবোর্ডে গিয়ে "OAuth & Permissions" সেকশনে যান। সেখানে নিচের Scope গুলো অবশ্যই অ্যাড করতে হবে:
chat:write (মেসেজ পাঠানোর জন্য)
chat:read (মেসেজ রিড করার জন্য)
৪. App ইন্সটল করা: একই পেজে (OAuth & Permissions) উপরে গিয়ে "Install to Workspace" বাটনে ক্লিক করতে হবে এবং পারমিশন এলাউ করতে হবে।
৫. Token সংগ্রহ ও ব্যবহার: ইন্সটল করার পর আপনি একটি Bot User OAuth Token পাবেন। মেসেজ পাঠাতে বা রিড করতে চাইলে এই টোকেনটি রিকোয়েস্টের Header এ পাঠাতে হবে।

Postman Request (Slack)

Method: POST
URL: https://slack.com/api/chat.postMessage
Header: Authorization: Bearer YOUR_TOKEN_HERE

Body (JSON):
{ 
  "channel": "C06JA4DGFssfsNF", 
  "text": "Hello from Slack Bot API!"
}
Telegram Messenger

Telegram Bot সেটআপ নিয়ম

১. BotFather ব্যবহার: টেলিগ্রাম অ্যাপে গিয়ে BotFather লিখে সার্চ করুন এবং নিচের কাজগুলো করুন:
• /start লিখে মেসেজ দিন।
• /newbot লিখে পাঠান।
• বটের একটি নাম (Bot Name) দিন।
• বটের একটি ইউজারনেম (Username) দিন।
২. Bot তৈরি ও URL: ইউজারনেম দেওয়ার পর আপনার বটটি ক্রিয়েট হয়ে যাবে। BotFather আপনাকে বটের একটি URL দিবে, সেই লিংকে ঢুকলে আপনি বটের মেসেজ দেখতে পাবেন এবং মেসেজ পাঠাতে পারবেন।
৩. User Data / Get Updates: ইউজার ইনফো বা আপডেট দেখতে নিচের লিংকে ব্রাউজ করুন (আপনার টোকেন সহ):
https://api.telegram.org/bot93727:AAGgwPUI7reTQrJ0_J2YxL5h9S8I/getUpdates
৪. Message Create URL: বট দিয়ে মেসেজ পাঠাতে এই এপিআই ব্যবহার করুন:
https://api.telegram.org/bot87212:AAG7ZZrAWqmPv9D1vg9SlpI9JnbZag/sendMessage
৫. Token in Header: টেলিগ্রামের ক্ষেত্রেও রিকোয়েস্ট করার সময় টোকেনটি হেডার (Header) হিসেবে পাঠাতে হবে।

Postman Request (Telegram)

Method: POST
URL: https://api.telegram.org/botYOUR_TOKEN/sendMessage

Body (JSON):
{
    "chat_id": "5204155030",
    "text": "Hello Akjm! I'm bot"
}
